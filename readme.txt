== Description ==
WPMapMaker plugin helps you to display a map styler editor on the front end. Your users can style google maps with WPMapMaker plugin from your website like Snazzymap. After styling the map they can easily copy the JSON object and passing this to the map options of any Google Maps object

You just need to use the <strong>[wpmapmaker]</strong> shortcode on a page display the map builder on your frontend.
